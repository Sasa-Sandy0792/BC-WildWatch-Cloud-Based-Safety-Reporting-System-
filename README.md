# BC-WildWatch-Cloud-Based-Safety-Reporting-System-

C WildWatch™ – Cloud-Based Safety Reporting System - - - - - - - 
Deployed a web-based campus safety application using Microsoft Power Platform (Power 
Apps, Power Automate, Power BI) 
Integrated Microsoft 365 (Azure AD / Entra ID) authentication for secure access control 
Developed interactive dashboards in Power BI for real-time incident reporting and analytics 
Implemented automation workflows, chatbot support, and alert systems 
Designed a secure, scalable cloud architecture using free-tier hosting platforms 
Contributed to deployment planning, risk assessment, and team coordination (RACI matrix) 
Enabled mobile access via QR code integration for live system demonstration
